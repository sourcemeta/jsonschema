Version
=======

```sh
jsonschema version
jsonschema --version
jsonschema -v
```

Print the current version of the JSON Schema CLI to standard output, without
the `v` prefix.

Examples
--------

### Print the current version

```sh
jsonschema version
```

### Print the current version using the long flag

```sh
jsonschema --version
```

### Print the current version using the short flag

```sh
jsonschema -v
```
